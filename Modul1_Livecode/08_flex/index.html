Was ist css flexbox

  • Flexboc ist eine moderne und einfache Möglichkeit, responsive und flexible Layoouts zu erstellen ==> wir brauchen keine festen Größenangaben
  • Das Modell arbeitet auf 2 Achsen auf denen die Inhalte verteilt werden
  • Flexbox ist für jede Bildschirmgröße geeignet
  • Flexbox arbeitet mit einem Containerelement(flex container) und den darin enthaltenen Kindelementen der 1. Ebene!!! -> flexbox wirkt genau eine Ebene tief -> weitere Verschachtelungen werden ignoriert
