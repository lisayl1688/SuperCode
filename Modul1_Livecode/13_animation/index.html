Transform, Transition, Animation

Transform

CSS transform ändet die position, die größe und die Form von HTML-Elementen

  • translate = bewegen
    • translate => entlang der x und Y Achse bewegen
    • translateX => entlang der X-Achse bewegen
    • translateY => entlang der Y-Achse bewegen
  • rotate = rotieren
    • rotate => um 0 bis 360 grad
    • rotateX => wir rotieren auf der X-Achse
    • rotateY => wir rotieren auf der Y-Achse
  • scale = skalieren
    • scale => vergrößern oder verkleinern
    • scaleX => vergrößern oder verkleinern auf der X-Achse
    • scaleY => vergrößern oder verkleinern auf der Y-Achse
