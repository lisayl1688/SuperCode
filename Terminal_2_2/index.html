CheatSheet - Terminal

mkdir -> Verzeichnis neu erstellen

cd -> Verzeichnis wechseln

ls -> kurze Auflistung

ls -l -> lange Auflistung

touch -> eine neue Datei erstellen

cp -> kopieren einer Datei

mv -> verschieben oder umbennenen einer Datei

rm -> entfernen einer Datei

clear -> löscht den Bildschrim

nano -> öffnet eine Datei mit dem nano-editor

tar -cvzf -> verpackt Dateien

tar -xvzf -> entpackt Dateien

cd .. -> eine Ebene nach oben
